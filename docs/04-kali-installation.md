# Kali Linux Installation

## ISO

Kali Linux ARM64 Installer

## Installation Steps

- Created ARM64 virtual machine
- Allocated 2 GB RAM
- Attached Kali ARM64 ISO
- Installed operating system
- Configured Adapter 1 as NAT
- Configured Adapter 2 as Host-Only
- Assigned static IP 192.168.56.10

## Issues Encountered

Initially downloaded the AMD64 installer which could not boot on Apple Silicon.

Resolution

Downloaded the ARM64 installer and repeated the installation successfully.
